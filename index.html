Sobre mim

I'm Front End Developer Jr.

Atualmente estou atuando como Desenvolvedor front-end React JS. Procuro estar sempre a par das novidades para criar interfaces seguindo conceitos modernos, responsivos e clean, focado na usabilidade.


Meus pontos fortes ressaltam em HTML, CSS, JavaScript, Bootstrap, Banco de dados(SQL) e UI Design, utilizando a ferramenta Figma, e principalmente venho crescendo a cada dia mais na biblioteca React JS.


Estou em constante aprendizado, participando de eventos e sempre programando e desenvolvendo algo.


Cursando Ciência da Computação UNIFACS. As conexões dentro da faculdade me possibilitaram ter minha experiência no mercado de trabalho na área de programação. Buscando novas habilidades na minha carreia, sou uma pessoa com grande senso de colaboração e trabalho em equipe, enfrentando desafios e criando aplicações web, desenvolvendo programas, aplicativos e sistemas.

Tecnologias

HTML5

HTML é uma linguagem de marcação utilizada na construção de páginas web

CSS3

O CSS é um mecanismo para adicionar estilo a um documento web

JavaScript

Uma linguagem de alto nível com tipagem dinâmica fraca e multiparadigma

React Js

Biblioteca usada para construir interface de usuário (UI)

Styled Components

Uma biblioteca que utiliza o conceito de CSS-in-JS, permite escrever códigos CSS dentro do Javascript.

React Native

Framework para desenvolvimento de aplicativos para Android e iOS de forma nativa.

Node Js

Executa códigos JavaScript no backend/servidor e APIs.

Figma

É um editor gráfico de vetor e prototipagem de projetos de design

Git

É um sistema de controle de versão distribuído de código aberto, desde projetos pequenos a muito grandes com velocidade e eficiência
